## Migrate Prisma
```
npx prisma migrate dev
```

## Name the Migration
```
npx prisma migrate dev --name init_schema
```

## Generate Prisma Client 
```
npx prisma generate --no-engine
```

## Add accelerate Extension
```
npm install @prisma/extension-accelerate
```


## Run Backend
```
npm run dev
```
